Add conditional functionality to Elementor Pro forms.

== Description ==

**Rocket Addons** is a well-designed, highly optimized, lightweight plugin that allows users to add conditional functionality to Elementor Pro forms. Rocket Addons it is one of the few plugins that allow you to add conditional functionality to hide or show form

fields in a simple manner, build a quoting calculator, or create custom values based on user selections.

Rocket Addons has an excellent customer service team that's ready to answer any questions that you have. Visit our support portal in the wordpress admin or contact us through our website.

[**Elementor PRO**](https://wordpress.org/plugins/elementor/) page builder is required for this plugin.

https://youtu.be/O2lghgjb3nE

###INSTALLATION

* First make sure that Elementor Pro Page Builder is installed, As this plugin works only with it.
* Download the plugin then Upload it to the plugin folder: /wp-content/plugins/ or install it through the WordPress plugins screen directly.
* Activate the plugin through the ‘Plugins’ screen in WordPress.
* You can find the plug-in under the category “Rocket Addons” on your Elementor element/widget list.

[Live Demo](https://rocket-addons.com/demos/)

###Extensions:
* [Conditional Visibility (Single Condition)](https://rocket-addons.com/conditional-visibility-single-condition/) - Hide or show Elementor Pro form fields based on a single condition.

###Extensions PRO:
* [IF THEN - Conditional Values](https://rocket-addons.com/if-then-conditional-values/) - Create IF, THEN conditions to output a value based on the user's selection.
* [Conditional Visibility (Multiple Conditions)](https://rocket-addons.com/conditional-visibility-multiple-conditions/) - Hide or show Elementor Pro form fields based on multiple conditions.
* [Total Cost Calculator](https://rocket-addons.com/total-cost-calculator/) - Create calculations and display an output on your Elementor Pro forms.

== Frequently Asked Questions ==

= Who should use Rocket Forms? =

Anybody who wants to extend Elementor's form builder and save money by eliminating 3rd party form builders by using our simple conditional logic functionality.

= Can I use the plugin without Elementor Page Builder? =

No. You must be using Elementor Pro in order to access Elementor's form builder as well as our enhancements.

= Does it work with any theme? =

Yes. It will work with anything that is compatible with Elementor.
